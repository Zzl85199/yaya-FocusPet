# 芽芽有點怕你 · 3D 🌱

原本的平面小遊戲全面升級成 **Three.js 3D 房間**,並拆成正式的多檔案專案。

## 玩法

| 操作 | 效果 |
|---|---|
| 拖曳空白處 | 旋轉房間視角 |
| 滾輪 / 雙指縮放 | 拉近拉遠 |
| 點小生物 | 摸摸(**有機率**掉扭蛋券,同時加好感度) |
| 點紅色扭蛋機 或「抽扭蛋」 | 用 1 張券抽飾品,蛋會真的滾出來彈開 |
| 「餵莓果」 | 生物會跳過去吃掉(**有機率**掉券 + 加好感度) |
| 「牙牙森林」/ 後牆那扇門 | 走出房間到**森林世界**探索、誘捕、撿蛋、帶寵物散步 |
| 「衣櫃」 | 查看圖鑑、穿脫飾品(依部位分類) |
| 「家族」 | 查看/切換已認識的家人,也可管理**寵物**與**孵蛋器** |

## 💗 好感度(牠有多怕你)

芽芽剛認識你時很怕生。好感度會改變牠跟你互動的方式:

- **怕你(好感度低)**:你一看牠,牠就「咻」地躲到盆栽旁或紙箱後面,只露出眼睛偷看你,而且很容易被嚇到。
- **熟一點(好感度中)**:願意小心翼翼靠近你,但還是會保持一點距離。
- **超黏你(好感度高)**:直接蹦到你面前討摸摸、開心跳、冒愛心,幾乎不會被嚇跑。

**摸摸 +2、餵莓果 +4** 好感度。跨過門檻時會提示牠「沒那麼怕你了」。扭蛋券改成**隨機掉落**(摸摸約 14~26%、餵莓果約 55%),不再是摸幾下就一定有。

## 👀 眼神感應(v1.1)

按「眼神感應」開啟鏡頭(影像只在本機分析、不上傳):
- **你看著螢幕** → 牠知道你在看,會跑到前面找你玩、盯著你的游標、開心蹦跳討摸摸
- **你轉頭 / 離開** → 牠開始自己玩:踢粉紅玩具球、追蝴蝶、原地轉圈、跑去盆栽旁睡午覺,還會**去撞積木塔、撲上豆袋彈跳、鑽進紙箱裡偷看**
- **突然回頭看牠** → 會被抓包!愣住:「被、被看到了……」然後不好意思地蹦過來

技術:face-api.js TinyFaceDetector,模型(約 190KB)直接內嵌在專案裡,離線可用。
不開鏡頭或權限被拒時,自動退回「游標感應」(游標 7 秒沒動=你不在看)。

字體換成 justfont 開源的 **jf open 粉圓體**(SIL OFL 授權),已裁切成遊戲用字內嵌。

## 三大新系統

1. **120+ 件扭蛋飾品**(不含扭蛋券)
   29 種基底造型 × 8 色系,加上 10 件獨一無二的傳說款。
   部位分成:頭上 / 臉上 / 脖子 / 背後 / 光環,每個部位各穿一件、可以疊搭。
   稀有度:普通 68% / 稀有 27% / 傳說 5%,抽到重複會退券。

2. **隨機家人來訪**
   每隔一段時間會「叮咚!」——芽芽的弟弟芽豆、妹妹芽莓、哥哥芽壯、
   芽媽、芽爸、芽奶奶會隨機來訪並換班登場。
   第一次見面 +2 券並永久解鎖,之後可在「家族」面板自由切換。
   衣櫃全家共用,誰出場誰穿。

3. **完整 3D 房間**
   Three.js 卡通渲染:木地板圓台、地毯、窗光、盆栽、
   看得到裡面彩蛋滾動的扭蛋機、柔和陰影、彩紙與愛心粒子。
   房間裡多了**積木塔、豆袋坐墊、紙箱**等會被互動的小道具,
   整體光線也調亮了(正面補光 + 提高曝光 + 霧往後推),畫面更清晰不昏暗。

## 🌲 牙牙森林 · 寵物 · 精靈(v1.2)

房間後面多了一扇門。點畫面上的「**牙牙森林**」按鈕、或直接點那扇門,牙牙就會**走出房間到森林世界**。森林是完全獨立的戶外場景(草地、水池、一圈樹林與石頭、一間小木屋),房間會在進入森林時隱藏、回來時恢復。

1. **房外的森林世界 + 一扇門**
   森林在房間之外,靠後牆的門進出。所有**野生生物都在森林裡**遇到與捕捉,不再出現在房間內。

2. **更多樣的寵物與精靈**
   - **寵物(13 種)**:cottonbun、pip、bubble、fluff、slowpoke、mochi、hoppy、toffee、spike、paddle、ember、starcat、crystalfox,涵蓋兔子/鳥/魚/雞/蝸牛/貓/青蛙/小狗/刺蝟/鴨子/狐狸等造型,分三種稀有度。**寵物會跟隨主人**(在房間與森林都跟著牙牙)。
   - **精靈(11 種)**:leaf、dew、pebble、sprout、bloom、moth、ember、frost、spark、moon、aurora,有會飛、會發光的變化。**精靈只待在家**(房間裡遊蕩),不會跟出門。

3. **莓果誘捕(有機率成功/失敗)**
   在森林裡選一隻野生寵物或精靈,丟出**莓果**誘捕。愈稀有的成功率愈低(普通 40% / 稀有 22% / 傳說 9%,詳見下方 v1.3 調整);**失敗牠就會逃走**。抓到寵物會加入你的隊伍,抓到精靈會回到家裡。

4. **選寵物去散步 → 進化**
   在「家族 → 🐾我的寵物」面板選一隻設為**散步夥伴**,帶牠進森林走動會累積散步距離。走得夠多會**進化**(幼生 → 成長 → 完全體),外型變大、長出冠羽/環繞光點等特效。

5. **牙牙視角(第一人稱)/ 第三人稱**
   在森林裡按鏡頭鈕可切「🐾牙牙視角」——**第一人稱**跟著牙牙的頭走(在森林開鏡頭只是切視角,**不會進入 focus 專注模式**);關掉則回到**第三人稱**環繞視角。第一人稱時拖曳可轉頭,第三人稱時拖曳可繞圈、滾輪拉遠近。

6. **撿蛋 → 孵化精靈**
   森林地上「**有機率**」出現蛋(不是每次去都有),點它撿回**孵蛋器**、蛋就會從地上消失。孵化條件每顆不同,詳見下方 v1.3。

## 🔧 遊戲平衡調整(v1.3)

依實際玩過的回饋做了幾個調整:

1. **蛋改成隨機出現、條件各自不同、可檢視內容**
   森林地上的蛋現在是「**碰運氣**」才會出現(同時最多 1 顆,每次骰約 35% 機率生成),不再保證每次去都有。撿到後蛋會**立刻從地上消失**,進入「家族 → 🥚孵蛋器」的清單,清單會列出每顆蛋的**專屬外觀色**與**孵化條件**,方便檢視。每顆蛋隨機抽到不同的孵化方式:
   - 🚶 森林散步距離
   - 🎯 Focus Mode 專注時間(要開著眼神感應、真的看著螢幕才算)
   - 🫐 在家餵莓果次數
   - ✋ 摸摸互動次數
   - 🕰️ 什麼都不做、單純把遊戲開著等待
   不再是每顆蛋都只能靠「等時間到」孵化。

2. **精靈誘捕成功率調低 + 每種都有專屬特徵**
   誘捕成功率從原本 70%/42%/18% 降到 **40%/22%/9%**(稀有度越高越難抓)。11 種精靈也不再只是換色而已,各自加了專屬造型特徵——葉葉精靈頭上頂葉子、露露精靈頭頂水珠、石石精靈背著石粒、芽芽精靈長兩根嫩芽、花花精靈有花瓣裙擺、蛾蛾精靈有觸角、燼燼精靈頭頂小火苗、霜霜精靈身上冰刺、電電精靈拖著閃電尾巴、月月精靈頭頂懸浮弦月、極光精靈拖著三色彩帶。

3. **家人解鎖條件不再是單純等時間,每位都不一樣**
   打開「家族」面板,還沒認識的家人卡片上會顯示**專屬解鎖進度條**:
   - 芽豆(弟弟)→ 森林探索累積時間
   - 芽莓(妹妹)→ 好感度
   - 芽壯(哥哥)→ 誘捕成功次數
   - 芽媽 → 在家餵莓果次數
   - 芽爸 → Focus Mode 專注時間
   - 芽奶奶 → 抽扭蛋次數
   條件達成才會「叮咚!」來訪並永久解鎖。已經認識的家人仍會偶爾隨機來串門子(純粹調劑氣氛,不影響解鎖)。

4. **出森林時鎖定當前角色,回家才能換人**
   在牙牙森林裡「家族」面板的換人功能會**暫時鎖住**(點了會提示「先回家才能切換」),家人來訪/自動換人也會暫停;回到房間後才能自由切換家族成員與角色。

5. **精靈越抓越多的原因**
   精靈只要抓到或孵化就會**永久待在家裡**、目前沒有數量上限,而舊版誘捕成功率高達 70%,幾乎丟一次莓果就會抓到,累積幾次遊玩下來自然越養越多。這次把成功率降到 40%/22%/9%,之後的誘捕會慢下來很多;已經養在家裡的精靈是你的收藏成果,不會被清除。

## 🆕 這次更新(v1.4)

1. **森林撿到的蛋會立刻消失(修正)**
   之前在森林點蛋撿起來,蛋卻一直留在地上——那是因為程式只找到蛋的其中一塊模型、沒把整顆蛋群組移掉。現在撿起後蛋會**立刻從地上消失**(還會冒一下白煙),正常進孵蛋器。

2. **家族成員各有專屬特色**
   每位家人都加上一眼能認出的道具:**芽奶奶**拄拐杖、戴圓眼鏡、披披肩;**芽爸**戴老花眼鏡、拿報紙(保留鬍子);**芽媽**圍圍裙、拿木勺;**芽壯**披英雄小披風、綁頭帶;**芽莓**綁蝴蝶結、抱愛心抱枕;**芽豆**戴會轉的竹蜻蜓帽、腳邊有玩具球。家族面板也會列出每位的特色。

3. **寵物進化變慢 + 多種進化方式**
   進化門檻大幅提高(散步 24/64 → 90/240),而且**每種寵物的進化方式不一樣**:散步🚶 / 餵莓果🫐 / 摸摸✋ / 一起 Focus🎯 / 靜靜陪伴🕰️。「我的寵物」面板會顯示每隻專屬的進化方式與剩餘進度。

4. **更多房間布景 + 隨機解鎖勳章掛牆上**
   房間多了彩旗、牆上時鐘、左牆層板(盆栽+書)、兩個相框,還有一面**勳章牆**。共 12 種勳章,玩遊戲時(摸摸/抽蛋/誘捕/孵蛋/進化/專注/認識家人/進森林)都有機率**隨機**掉勳章,解鎖後會**掛到後牆的勳章板上**,也能在「家族」面板檢視。

5. **牙牙的小祕密 / 小故事(隨機解鎖 + 記錄)**
   底部新增「📖 祕密」面板。在**森林散步**或**Focus Mode 專注**時,有機率**隨機**發現一則牙牙的小祕密、心得、小故事或冷知識(共 32 則),發現後永久記錄在祕密面板裡。

除錯小抄新增:
- `YY.tryRandomMedal(1)` 立刻隨機解鎖一個勳章,`YY.medalsOwned` 看清單
- `YY.tryUnlockSecret(1,'forest')` 立刻隨機解鎖一則小祕密,`YY.metSecrets` 看清單
- `YY.PET_EVO` 看每種寵物的進化方式,`YY.addEvoProgress('berry',10)` 直接餵當前寵物進度

## 執行方式

- **最簡單**:直接用瀏覽器打開 `dist/yaya3d-bundle.html`(需要網路載入 three.js)。
- **開發**:在專案根目錄跑任一靜態伺服器,例如 `npx serve .`,再開 `index.html`。
- **重新打包**:`node tools/build.js`

進度(券數、圖鑑、穿搭、認識的家人、好感度)都會自動存在瀏覽器 localStorage。

## 除錯小抄

打開瀏覽器主控台:
- `YY.familyVisit('meimei')` 立刻讓芽莓來訪
- `YY.attention.watching` 看目前判定你有沒有在看
- `YY.addTickets(50, '作弊')` 加券
- `YY.bumpTrust(60)` 灌好感度(想直接看「超黏你」的樣子)
- `YY.trust`、`YY.trustTier()` 看目前好感度與分級
- `YY.knockBlocks()` 手動撞倒積木塔
- `YY.setMode('explore')` / `YY.setMode('interact')` 進出森林
- `YY.toggleForestFPV()` 切牙牙視角(第一人稱)/第三人稱
- `YY.addPet('starcat')`、`YY.setActivePet(uid)` 直接給寵物 / 設散步夥伴
- `YY.addWalkToActivePet(60)` 灌散步距離看進化
- `YY.addEgg()` 撿一顆蛋(會隨機分配孵化條件),`console.log(YY.eggs)` 看每顆蛋的 `cond`/`need`/`prog`
- `YY.eggProgressWalk(200)` / `YY.eggProgressFocus(200)` / `YY.eggProgressBerry()` / `YY.eggProgressPat()` / `YY.eggProgressTime(200)` 直接灌對應條件的進度、快轉孵化
- `YY.familyUnlockProgress('doudou')` 看某位家人的解鎖進度(還沒認識的家人都適用)
- `YY.ownedPets`、`YY.homeSpirits`、`YY.eggs` 直接看狀態
- `YY.tickets`、`YY.owned` 直接看狀態
